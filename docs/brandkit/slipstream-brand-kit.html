08 USAGE GUIDELINES

How to use the mark

DO
  • Always write "slipstream" in lowercase, including at sentence start
  • Restructure sentences to avoid starting with the name when possible
  • Use s// as the compact icon mark
  • Pair with Geist Mono for all mark/code contexts
  • Use the Undertow gradient for accent elements
  • Reference as a sibling to mish, not a sub-product
  • Include the s// glyph in the wordmark: s//slipstream
DON'T
  • Never capitalize: "Slipstream" or "SLIPSTREAM"
  • Never use mish's violet-blue "Spectral Shift" gradient
  • Never describe as "a mish plugin" or subordinate tool
  • Never use marketing language: "blazing," "revolutionary," "game-changing"
  • Never use serif fonts in any context
  • Never separate the s// from the wordmark with a space: s// slipstream
  • Never use a light background as default. Dark is the org standard.
